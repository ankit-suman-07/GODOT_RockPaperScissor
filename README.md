## 💻 Setup locally

**Prerequisites** :
- Git 
- Godot v4.6.1

**Step 1:** Clone the repository
```
git clone https://github.com/ankit-suman-07/GODOT_RockPaperScissor.git
```

**Step 2:** Open Godot
```
In godot editor, click open a new project and browse to the cloned repository folder and select.
```
